JSwitchable
===========

a greate jquery plugin wrapper for yii community,you can easily do image switching




## Basic Markup Structure

    <div id="container">
      <div>First Panel</div>
      <div>Second Panel</div>
      <div>Third Panel</div>
    </div>


## Initialize Switchable

    <script>
    $(function(){
      $('#container').switchable();
    });
    </script>
